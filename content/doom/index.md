## Setup guide
Link to project: [github.com](https://github.com/andreyvelsk/com.mobilerpgpack.phone)

- Download the latest .apk file from GitHub.
- Add your own .WAD files.
- Configure the controls to your liking.

## Known issues

- At the start of the game, you can see the following issue:

> failed to create "....\uzdoom\config\uzdoom" folder

Just create this folder manually, or set another path in the `Paths to user folder` option.
