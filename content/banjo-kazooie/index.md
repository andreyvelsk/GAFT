## Setup guide
Link to project: [github.com](https://github.com/AurelioB/BanjoRecomp-Android)

- Download the latest .apk file from GitHub.
- Add your own ROM file. You need the 1.0 ROM version.

## Tips
- The port has mod support, such as [BK Reloaded texture pack](https://thunderstore.io/c/banjo-recompiled/p/GhostlyDark/BK_Reloaded/). Other mods have not been tested by the author. For extra mod info, visit the original [BanjoRecomp](https://github.com/BanjoRecomp/BanjoRecomp#mod-support) project page.
