## Description
Here is the description from the source:


1. Steam version running through GameNative.

2. Graphics settings set to the lowest preset, with the physical controls mapped to keyboard and mouse inputs using GameNative's quick menu.

3. Workshop mods downloaded and installed through GameNative's menu.

4. Custom GPU drivers and a CPU underclock for better thermals and battery life.

5. On the lower screen, I'm running the Civilopedia app from the Google Play Store, which is incredibly useful for quickly checking units, technologies, civ abilities, and game mechanics without leaving the game.
